# 電話占いリアルタイム補助ツール 設計書

## 概要

電話占いの通話中に、相手の音声から感情・状態を分析し、
「今どう鑑定を進めるべきか」をリアルタイムで画面に表示する補助ツール。

---

## アーキテクチャ全体図

```
[通話音声（マイク入力）]
        ↓
[OpenAI Realtime API]
  - 音声ストリーミング受信
  - 感情分析（不安/興奮/落ち込み/迷い 等）
  - テキスト起こし（同時）
        ↓
[分析結果をClaudeに送信]
  - 感情状態
  - 発言内容（テキスト）
  - 相談テーマの推定（恋愛/仕事/家族 等）
        ↓
[Claude API（鑑定アドバイス生成）]
  - 今の相手の状態サマリー
  - 次に聞くべきこと
  - 鑑定の方向性（四柱推命の何を見るか）
  - 地雷になりそうなポイント
        ↓
[画面表示（サブウィンドウ）]
  - PCで通話しながら横に表示
```

---

## 使用技術

| 役割 | 技術 | 備考 |
|------|------|------|
| 音声キャプチャ | PyAudio / sounddevice | マイク入力をリアルタイムで取得 |
| 感情分析 + 文字起こし | OpenAI Realtime API（gpt-4o-realtime） | 声のトーン・ピッチ・感情を直接解析 |
| 鑑定アドバイス生成 | Claude API（claude-sonnet-4-6） | 占い師視点のプロンプトを組み込む |
| UI | Python（tkinter / rich） | ターミナル or 軽量GUIウィンドウ |
| 言語 | Python 3.11+ | |

## データフロー詳細

### Step 1: 音声キャプチャ
```python
# マイク音声を 100ms チャンクで取得し続ける
# 通話がPCベース（LINE/Zoom等）の場合はシステム音声も取得可能
```

### Step 2: OpenAI Realtime API に投げる
```
送信: 音声ストリーム（PCM 16bit, 24kHz）
受信:
  - transcript: 発言テキスト
  - emotion: { primary: "anxiety", confidence: 0.82 }
  - tone: { pace: "slow", pitch: "low", energy: "low" }
```

### Step 3: Claude API へのプロンプト構成
```
[システムプロンプト]
あなたは四柱推命の占い師（長谷川あつき）の鑑定補助AIです。
鑑定実績1,000件以上のプロ占い師を補助します。
以下の情報をもとに、今この瞬間にすべきことを簡潔に提示してください。

[ユーザープロンプト（自動生成）]
## 相手の状態
- 発言: 「仕事のことが不安で…なかなか決断できなくて」
- 感情: 不安（82%）/ 迷い（61%）
- トーン: 話すペースが遅い、声が低い、エネルギーが低め

## 質問
1. 今の相談者の状態を一言で
2. 次に聞くべきこと（1つだけ）
3. 四柱推命で見るべき柱・十神
4. 注意すること
```

## コスト試算

| API | 単価 | 1時間の通話あたり |
|-----|------|-----------------|
| OpenAI Realtime API | 約$0.06/分（音声入力） | 約$3.6（約550円） |
| Claude API（Sonnet） | 約$0.003/1Kトークン | 約$0.3（約45円）※5秒ごと更新で試算 |
| **合計** | | **約600円/時間** |

電話占い1時間の収益（時給3,000〜8,000円）に対してコストは約600円。
**収益の10〜20%程度のコストで補助精度が大幅に上がるなら十分ペイする。**
