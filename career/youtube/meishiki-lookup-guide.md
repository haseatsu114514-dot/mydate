# 【実演】マニアック四柱推命で命式を調べる方法

## この台本の位置づけ

- メイン動画（astrology-intro-video.md）の **Part 0「視聴者への準備」（3:00〜5:00）** で使う実演パート
- 単体でもShorts / 短尺動画として切り出せる構成にしておく
- 視聴者が「自分で命式を出せた！」という成功体験を得ることがゴール

「まず『マニアック四柱推命』って検索してもらえればすぐ出てきます。はい、これですね。トップページが開いたら、上のメニューのところに『命式作成』ってあるんで、そこをポチッと押してください。はい、これで入力画面にいけました」

[画面: Google検索で「マニアック四柱推命」と入力 → サイトにアクセス → 「命式作成」メニューをクリック]

### ステップ2：生年月日を入力する（0:30〜1:00）

「入力するのはたったの3つだけです。生年月日、生まれた時間、性別。これだけ。はい、じゃあ実際にやってみますね。今回は例として、僕の誕生日1997年1月19日を入れてみます。はい、1997年、1月、19日っと」

[画面: 生年月日の欄に 1997年1月19日 を入力]

「で、次に生まれた時間なんですけど、これ、わからない人結構いると思うんですよ。母子手帳に書いてあったりするんですけど、見つからないって人は全然"不明"で大丈夫です。時間がわからなくても日柱はちゃんと出るんで、今回の相性鑑定には全く問題ありません。はい、性別を選んで、『作成』ボタンをポチッと」

[画面: 生まれた時間を入力（または「不明」を選択） → 性別を選択 → 「作成」ボタンをクリック]

「あ、そうそう大事なこと言い忘れてた。自分の分だけじゃなくて、気になる相手の分も同じように出してくださいね。2人分必要です」

[画面: 命式表が表示される。日柱の部分を赤枠で強調]

天干     ○      ○     【ここ】   ○

「で、この上の天干のことを日干（にっかん）って呼ぶんですけど、これがあなた自身を表す、四柱推命で一番大切な一文字です。まずこの日干をメモしてください」

[画面: 日干の部分を矢印で指し示し、「日干 = あなた自身」のテロップ表示]

「僕の場合は、日干が壬（みずのえ）で、日支が寅（とら）ですね。壬寅（みずのえとら）。はい、こんな感じでメモっておいてください」

[画面: 日柱の「壬」「寅」をそれぞれハイライト表示]

## 制作メモ

### 画面収録のポイント

- **実際のサイト画面を録画する**（スクリーンキャプチャ）
- マウスカーソルの動きをゆっくり、大きく見せる
- 入力するたびに一呼吸おく（視聴者が追いつけるように）
- 命式が表示されたら **3秒間はそのまま映す**（スクショタイムを作る）

### 編集で入れるもの

- 日柱を赤枠で囲む（After Effects or Premiere Proのマスクでも可）
- 天干・地支のラベルをテロップで追加
- 「ここだけ見ればOK！」のテキストを画面にバーンと出す
- 矢印で「日干 = あなた自身」を指し示す

### 概要欄に載せるリンク

```
■ 命式を調べる（マニアック四柱推命）
[URLをここに記載]

■ 命式の出し方がわからない方
→ この動画の 3:00 から実演しています
```

### Shorts切り出し

- **「30秒で命式を出す方法」** として切り出し可能
  - サイトアクセス → 入力 → 日柱を確認、の流れをテンポよく
  - 最後に「詳しくは本編で！」と導線
- **「命式で最初に見るべきたった1つの場所」**
  - 日干だけにフォーカスした15秒〜30秒の切り出し

### 注意事項

- 著作権：マニアック四柱推命の画面をそのまま使う場合、サイトの利用規約を確認すること
- 他の命式サイト（例：精密四柱推命、命式自動計算など）でも同じ手順で出せる旨を一言添えると親切
- 個人情報：実演で使う生年月日は自分のものを使用（1997年1月19日）
