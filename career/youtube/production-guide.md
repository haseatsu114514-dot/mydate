# YouTube動画 制作ワークフローガイド

四柱推命チャンネルの動画を効率よく量産するためのガイド。
AI活用 + Premiere Pro で、なるべく手を動かさずにクオリティを保つ。

---

## 1. スライド量産の方法（一枚ずつ作らなくていい）

### Gamma (gamma.app) — 台本からスライド自動生成

- テキストを貼るだけでスライドデッキを自動生成してくれるAIツール
- 無料枠あり、日本語対応
- **使い方**: 台本のセクションごとにテキストをコピペ → 「Generate」→ デザイン調整 → PNG/PDFエクスポート
- 四柱推命の解説パートなど、テキスト中心のスライドに最適
- エクスポートしたPNGをPremiere Proのタイムラインに並べるだけ

### Canva + 一括作成 — テンプレ×データで量産

- テンプレートを1枚作り、「一括作成（Bulk Create）」でCSV/スプレッドシートからデータを流し込む
- **具体例**:
  - 十干（甲〜癸）の解説スライド10枚 → テンプレ1枚 + CSVに干の名前・キーワード・イメージカラーを入れる
  - 干合5パターン → 同様にテンプレ1枚 + データ5行
  - 十二運、通変星なども同じ要領
- 手順: テンプレ内のテキストを「{{列名}}」に置換 → データ接続 → 一括生成 → PNG一括ダウンロード

### Google Slides + Apps Script — プログラムで差し替え

- テンプレートスライドを1枚作成
- Google Apps Scriptでプレースホルダーをデータ配列から差し替えて量産
- Claude Codeにスクリプトを書いてもらえる（スプレッドシートのデータ → スライド生成）
- Canvaの一括作成で足りない複雑なレイアウトの場合に有効

### 今回の動画に必要なスライド × 量産方法

| スライド種類 | 枚数目安 | おすすめ方法 |
|---|---|---|
| タイトル・導入 | 1-2枚 | Canvaで手作り |
| 用語解説（十干・十二支など） | 10-12枚 | Canva一括作成 |
| 図解（五行相関図など） | 2-3枚 | Claude SVG生成 → Canva取込 |
| 事例・具体例 | 3-5枚 | Gamma自動生成 |
| まとめ・CTA | 1-2枚 | Canvaで手作り |

## 2. AI活用で楽できるポイント

### 台本生成
- Claude Codeで台本を書く（もうやってる）
- 構成案 → セクションごとの台本 → スライド指示書まで一気に出せる

### サムネイル
- **Canva AI**: テンプレから生成、文字入れまで完結
- **Midjourney / DALL-E**: 四柱推命っぽいビジュアル素材を生成（和風・陰陽・五行イメージ）
- 毎回ゼロから作らず、サムネテンプレを2-3パターン用意して回す

### テロップ自動生成
- **Premiere Pro 自動文字起こし**: 音声トラック → 「テキスト」パネル → 「文字起こし」→ キャプション生成 → スタイル適用
- **Vrew**: 無料。動画を読み込むだけで自動テロップ。Premiere連携も可。手軽さはこっちが上
- どちらも精度は高いが、専門用語（四柱推命、通変星など）は手動修正が必要

### BGM
- **Suno AI / Udio**: 「和風 穏やか 占い BGM instrumental」などのプロンプトで生成
- 著作権フリーで使える（利用規約は要確認）
- 数曲生成してストックしておけば使い回せる

### ナレーション補助
- 自分で喋る前提だが、仮ナレーションでタイミング確認に使える
- **VOICEVOX**（無料・日本語特化）: 台本を流し込んで仮音声生成 → タイムライン上で尺感を確認
- **ElevenLabs**: より自然な音声が必要な場合

### 図の生成
- Claude / ChatGPTに五行循環図・相生相剋図・命式チャートのSVGを生成させる
- SVGをそのままCanvaにアップロードして色やフォントを調整
- 一度作った図はテンプレとして再利用

## 3. Premiere Proでの効率的な編集フロー

### テンプレートプロジェクト
- `.prproj`のテンプレートを1つ作っておく
- 含めるもの: シーケンス設定（1080p/30fps）、BGMトラック配置、テロップスタイル、カラー設定
- 毎回このテンプレを複製して作業開始

### テロップ統一
- エッセンシャルグラフィックスでテロップテンプレート（.mogrt）を作成
- フォント・サイズ・背景色・位置を統一
- 「通常テロップ」「強調テロップ」「用語テロップ」の3種類あれば十分

### 自動文字起こし → テロップ化の流れ
1. 音声収録済み素材をタイムラインに配置
2. 「テキスト」パネル → 「シーケンスから文字起こし」（日本語選択）
3. 文字起こし結果を確認・修正（専門用語を直す）
4. 「キャプションの作成」→ スタイル適用
5. 必要に応じてキャプションをグラフィックに変換して位置調整

### Shorts切り出し
1. 本編シーケンスを複製
2. シーケンス設定を1080×1920（9:16）に変更
3. 素材の位置・スケールを調整
4. 見どころ30-60秒を切り出し
5. 別書き出し → Shortsとして投稿

## 4. 具体的な制作スケジュール案

| 日 | 作業内容 | 所要時間目安 |
|---|---|---|
| **Day 1** | 台本確定（Claude Code）＋ スライド生成（Canva一括/Gamma） | 2-3時間 |
| **Day 2** | 収録（OBSでスライド画面共有 + 喋り） | 1-2時間 |
| **Day 3** | 粗編集（Premiere自動文字起こし → テロップ、不要部分カット） | 2-3時間 |
| **Day 4** | 仕上げ（BGM・SE挿入、テロップ修正、色調整） | 1-2時間 |
| **Day 5** | 公開準備（サムネ作成、概要欄、タグ、予約投稿） | 1時間 |

**合計: 約7-11時間/本**

慣れてきたらDay 1-2を同日にまとめて3-4日サイクルに短縮可能。

## 5. 収録方法の選択肢

### A. 画面収録＋声（最も手軽・おすすめ）
- OBS Studioでスライドを全画面表示 → 画面キャプチャ + マイク入力で録画
- 編集量が少なく、スライドの切り替えタイミングも自然に決まる
- **まず1本目はこれで出す。ハードルが一番低い**

### B. 顔出し＋スライド（PiP）
- カメラで自分を撮りつつ、スライドをPicture in Pictureで配置
- 信頼感・親近感が上がる。占い師としてのブランディングに有効
- 編集工数は増えるが、Premiere Proのマルチカメラ編集で効率化可能

### C. 顔出しなし＋スライド＋声
- スライドと声だけ。編集が最も楽
- 占い系チャンネルは顔出しなしでも十分伸びるジャンル
- Aとほぼ同じだが、OBSではなくスライドPNG + 音声を直接Premiereに並べる方法もある

### おすすめの進め方
1. **1本目**: A（画面収録＋声）で最速で公開
2. **2-3本目**: Cで編集フローを固める
3. **慣れてきたら**: Bに挑戦して差別化
