# mydate

自分の人生データを管理するリポジトリ。
キャリア相談・タスク管理・恋愛相談など、Claudeに相談する時の文脈として使う。

## 構成

```
mydate/
├── CLAUDE.md              ← Claudeが最初に読むファイル（全体サマリー）
├── profile/
│   ├── timeline.md        人生年表
│   └── personality.md     性格・価値観
├── career/
│   ├── current.md         今の仕事
│   └── direction.md       キャリアの方向性
├── skills/
│   └── inventory.md       スキル一覧
├── life/
│   ├── romance.md         恋愛の状況
│   └── tasks.md           やることリスト
└── goals/
    └── current.md         目標と振り返り
```

## 使い方

1. 各ファイルの `（記入）` 部分を自分の情報で埋める
2. 状況が変わったら随時更新する
3. Claudeに相談する時、このリポジトリをコンテキストとして渡す

## 注意

- 個人情報の粒度に注意（publicリポジトリの場合は特に）
- 具体的な金額・住所詳細・相手の実名は書かない
